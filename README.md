 # Skin Biome Explorer

An interactive data visualization tool for exploring the human skin microbiome.

[Live Demo](https://elinamrachkovska.github.io/Skin-Biome/)


## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/ElinaMrachkovska/Skin-Biome.git

# Navigate to the project folder
cd Skin-Biome

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
